VerifyYou SDK Playground

Drives @verifyyou-sdk/client straight from source. Enter a publishable key, hit Run vycheck() — you'll redirect to the hosted flow and land back here with ?vyc & ?vyt, which vyget() reads below. 

—


    
Usage

    
TypeScript / bundler:

    
import { init, vyget, vycheck } from "@verifyyou-sdk/client";

init({ publishableKey: "pk_live_…" });    // once, at startup
const { verified, token } = vyget();      // after returning from the flow
if (!token) await vycheck();              // otherwise start + redirect

    
Raw <script> (UMD global Vy):

    
<script src="https://unpkg.com/@verifyyou-sdk/client"></script>
<script>
  Vy.init({ publishableKey: "pk_live_…" });
  if (!Vy.vyget().token) Vy.vycheck();
</script>
